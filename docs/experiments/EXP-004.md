## Frozen question

Can a global all-six-slice replay-tangent projection change the AdamW update
direction enough to preserve the unchanged cumulative replay ceiling and reopen
the quantity-learning path?

EXP-003 showed that scalar continuation was insufficient. Q2.5 accepted 66
full-scale and five backtracked updates, including one at 1/128, then exhausted
all eight scales on attempts 72–79. The terminal minimum-scale candidates missed
the 1.5% authority by only 0.000151–0.000238 percentage points. EXP-004 changes
direction, not the standard.

## Frozen intervention

At each committed pre-attempt state, the implementation computes the arithmetic
mean gradient `r` of the same six frozen replay validation windows used by the
cumulative guard. For each registered Q2.5 learning-rate scale, it forms the
scaled AdamW weight displacement `d` and applies

```text
d' = d - max(0, dot(r,d) / dot(r,r)) r
```

as one global projection over all trainable weights. A non-positive dot product
leaves the candidate unchanged. Projection changes weights only; the selected
scaled trial's AdamW moments commit with them. Every retry or rejection restores
weights and both moment arrays together.

The tangent condition is candidate construction, never experimental authority.
Every trial must still pass the unchanged direct six-slice functional evaluation
against immutable ZERO.3. The first finite trial at or below 1.5% commits.
