## Frozen intervention

At each committed pre-attempt state, the implementation computed the arithmetic
mean gradient `r` of the same six frozen replay validation windows used by the
cumulative guard. For each registered Q2.5 learning-rate scale, it formed the
scaled AdamW weight displacement `d` and applied

```text
d' = d - max(0, dot(r,d) / dot(r,r)) r
```

as one global projection over all trainable weights. A non-positive dot product
left the candidate unchanged. Projection changed weights only; the selected
scaled trial's AdamW moments committed with them. Every retry or rejection
restored weights and both moment arrays together.

The tangent condition constructed candidates but never supplied experimental
authority. Every trial still had to pass the unchanged direct six-slice
functional evaluation against immutable ZERO.3.
