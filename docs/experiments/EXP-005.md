under the unchanged gates?

## Frozen family rule

The Q2.6 family resolves `go` if and only if seeds 1, 2, and 3 each resolve
`go`. Aggregation waits for both new results in every case. If the conjunction
passes, the already-selected seed-2 quantized model with SHA-256
`44b32f2262be2754fd2eeaf16ed206bae32b4ce30d7f5541a1059cd21257ae50`
is the only ZERO.4 promotion candidate. Replication checkpoints cannot replace
it through post-hoc selection.

If either replication seed is no-go, the family is no-go and ZERO.3 remains
current.
