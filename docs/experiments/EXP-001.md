## Question

Can an observer-calibrated per-attempt functional replay budget prevent
faculty training from exceeding the frozen 2% public replay ceiling without
weakening the quantity gates?

The upstream Q2.3 contract was frozen before execution. ilXyr imported the
completed result through its retro lane because the run itself did not pass
through ilXyr's forecast and allocation ceremony. The registration is
therefore grounded but correctly records `forecast_risked=false`.

## Frozen design

The 4,852,992-parameter student, ZERO.3 initialization, three immutable
teachers, quantity corpus, replay corpora, routing boundary, seed, public
thresholds, 25-update recovery cadence, and 100-update full-evaluation cadence
were unchanged from Q2.2. Seeds 1 and 3 and the promotion split stayed sealed.

An observer run first proved that instrumentation was a learned-state no-op:
200 attempts and 200 commits were byte-identical to an unguarded reference at
every recovery. It calibrated a 0.1641% warning band and 0.25% hard local band.
The first-order predictor had Pearson correlation 0.0076 with realized replay
change, so it remained diagnostic and projection stayed disabled. The guarded
run used the direct functional replay probe as its only authority.

## Result

| Gate | Result | Decision |
| --- | ---: | :---: |
| Optimizer attempts | 200 | — |
| Commits / rejections | 200 / 0 | mechanics pass |
| Warning / hard exceedances | 5 / 0 | guard did not bind |
| Largest local replay increase | 0.2013% | below 0.25% hard band |
| Quantity operation rate | 95.0% | pass |
| Quantity exact-artifact rate | 95.0% | pass |
| Public replay regression | 2.685% | **fail** |
| Promotion evaluations | 0 | sealed |

The declared stop fired after replay exceeded 2% on two consecutive full
evaluations. There was no jointly feasible checkpoint and no selected model.
This rejects the independent local-budget hypothesis; it does not invalidate
the transaction, rollback, checkpoint, or replay-instrumentation mechanics.

```bash
cargo run -p ilxyr-cli -- init /workspace/ledger
cargo run -p ilxyr-cli -- retro /workspace/ledger \
  examples/families/zero-q23-seed2.retro.json --execute
```

The complete `retro --execute` path was exercised in a fresh local workspace.

The published spec uses container paths under `/workspace`; local users should
substitute absolute paths to their clean ilXyr and Zero checkouts without
changing the source commit, artifacts, metrics, seeds, or authority.
